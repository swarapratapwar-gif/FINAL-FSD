Frequently Asked Questions

Everything you need to know about the EduAchieve platform.

🔍
Only registered students and faculty from PCCOE Pune with a valid @pccoepune.org email address can create an account and submit projects. However, viewing projects is open to the public.
For the mandatory Research Paper, only PDF (.pdf) files are accepted. For the optional presentation slides, you can upload PowerPoint files (.ppt, .pptx). The maximum file size limit is 20MB per file.
Currently, projects cannot be edited by students after submission. If you made a mistake or need to update your files, please contact the department admin or your project guide to request a modification.
EduAchieve strictly enforces the PCCOE email pattern: firstname.lastname##@pccoepune.org. If you are using a personal Gmail account or a differently formatted college email, the system will reject it.
No, we do not host source code directly to save server storage. Instead, you should provide a link to your public GitHub repository during the submission process.
You can filter projects by Batch, Year, and Domain simultaneously. The search bar at the top of the sidebar provides full-text search across project titles and team member names.
Currently, self-service password reset is not automated. Please contact the platform admin (admin@pccoepune.org) from your registered college email address to request a temporary password.
During submission, there is a dedicated "Team Members" text area. Simply type the full names of all your team members, separated by commas (e.g., "John Doe, Jane Smith").
The Guide Name refers to the PCCOE faculty member who supervised or mentored your academic project. This is a mandatory field for all submissions to ensure academic authenticity.
Projects appear on the portal immediately after submission. However, department admins regularly monitor the feed and will delete any inappropriate, plagiarized, or invalid submissions.
