CSS Variables Theme Toggle

This page demonstrates how to implement a robust dark/light theme toggle using CSS Custom Properties (variables) and localStorage.

"The best way to implement dark mode is by relying entirely on CSS Custom Properties and avoiding inline styles or multiple stylesheets."

How it works

We define all our colors as variables inside the :root pseudo-class. Then, we create a .dark-theme class that overwrites those specific variables.

/* Define default variables */
:root {
  --bg-color: #FFFFFF;
  --text-color: #333333;
}

/* Overwrite variables for dark mode */
body.dark-theme {
  --bg-color: #121212;
  --text-color: #EEEEEE;
}

/* Use variables throughout the CSS */
body {
  background-color: var(--bg-color);
  color: var(--text-color);
  transition: all 0.3s ease;
}
