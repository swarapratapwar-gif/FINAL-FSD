✨

What this project does

A student-focused portal where users can register, log in, submit final-year work, and browse project submissions with owned profile links.

📚

Core features

Project submission, searchable project browsing, student records, profile pages, notifications, and file uploads for research papers and presentations.

🔒

Why it is reliable

JWT authentication, protected actions, a file-backed store for stability, and a shared frontend layout that keeps the experience consistent.
